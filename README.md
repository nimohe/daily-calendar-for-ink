# Zen Daily Calendar

一个为墨水屏设备优化的日历应用，支持小米电纸书1代及 Android WebView 61+ 浏览器。

## 功能

- 📅 显示公历日期、农历日期、星期
- ⏰ 实时时间显示（两种样式可切换）
- 💬 每日格言
- 📱 响应式布局，兼容竖屏/横屏
- 🖥️ 兼容旧版浏览器（Android WebView 61+）

## 技术栈

- React 19
- Tailwind CSS 3
- Vite 6
- Motion（动画）
- lunar-javascript（农历计算）

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物在 `dist/` 目录。
